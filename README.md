# Book Explorer

**Book Explorer** is a web application built with Python, designed to help users search for books using the Google Books API and create a personal account to save books of interest. The application combines interactive search functionality with a personalized library, making it easy to discover and manage favorite books.

---

## Features

- **Search Books**: Query the Google Books API to find books by title, author, or keyword.  
- **User Accounts**: Create and manage personal accounts to save favorite books.  
- **Personal Library**: Add, view, and remove books from your own collection.  
- **Responsive User Interface**: Built with Flask and Bootstrap for a smooth and mobile-friendly experience.  
- **Data Storage**: User information and saved books are stored using SQLite, ensuring persistent and reliable data management.  

---

## Technology Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, Bootstrap  
- **API:** Google Books API  
- **Database:** SQLite
